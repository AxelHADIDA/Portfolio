Projets Personnels Encadrés

Au cours des deux ans de BTS, deux projets ont été mis en place. 
Nous avons choisi de développer un site web et une application de gestion pour l'entreprise fictive Auto Ecole Castellane, dont l'activité principale est la location de véhicule auto école et exament du code et permis de conduire. 

sp1 - site web auto école castellane

Le site internet est une vitrine pour L'auto école Castellane. Il permet aux moniteurs de créer un compte propriétaire afin de proposer des heures de conduite aux clients ou un compte client afin de se faire contacter pour la réservation d'heures de conduite choisi sur le site.

Compétences associées : A1.1.1,A1.2.2, A1.4.1, A2.3.1,A4.1.1, A4.1.2, A4.1.3, A4.1.6, A4.1.7, A4.1.9, A4.1.10.

sp2 - application android myauto

Pour ce projet, nous avons utilisé les langages Java sous Android Studio et SQL. 
Le projet a été piloté avec Gantt project. Le logiciel de versionning utilisé est GitHub. 

Compétences associées : A1.1.1, A1.4.1, A4.1.1, A4.1.2, A4.1.3, A4.1.4, A4.1.9, A4.1.10.
